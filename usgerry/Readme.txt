This app can be run easily at
https://michaelarnold.shinyapps.io/usgerry/

Use the Picker tool to set which State you are painting with, or use the drop down.  
Three extra states are available, as well as a 'none' option to remove a county.

Party numbers are simply taken from the 2020 presidential election - this is not entirely predictive.

Be careful painting an entire state - there is currently no undo button!
